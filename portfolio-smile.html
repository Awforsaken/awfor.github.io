Revamp of Smile Banking mortgages landing page

Included Paper sketches, Wireframes, High Detail Design, UX

Introduction

Getting mortgage is a big and scary commitment. In the land of internet, this could be worked around by creating a welcoming, clear and friendly atmosphere and this is what I did with this redesign of Smile banking mortgages landing page.

Idea was to revamp of Smile banking mortgages landing page and make it more assuring and friendlier without massively overhauling entire page.

Since such thing as desktop-only design is a thing of the past, I have done this for 3 different view ports desktop, tablet, and mobile but will focus on the design process of desktop here.

Initial analysis

The first step was to analyze the current page and identify key shortcomings. Since I did not have access to analytics I kinda had to rely on obvious issues and guesswork here. Key issues:

  • Lacks content.
  • No user sorting.
  • Banner could be adapted to be functional and provide useful information.
  • Very limited colour palette - makes the page look as though it all just melts into one.
  • Quick user journey.(in a sense that there is hardly any)

Now not all things were that bad, some of the elements and solutions used here were still salvageable.

  • Buttons are highlighted to attract user attention, as they are the main content of this page.
  • Interactive(had .hover interaction, not much but still something)
  • Minimal - easy to use and not confusing in layout.
  • Language sticks to brand well.
  • Interface design is consistent.
  • Quick user journey. Hardly any distractions

Competition Analysis

Mortgages is a fairly key part of most banks and there is no shortage for fairly up to date websites to draw inspiration from. So I compared Smile mortgages landing page to these:

  • NatWest
  • HSBC
  • Egg(Yorkshire building society)
  • Barclays
  • Halifax
  • Lloyds
  • Cooperative Banking

Current popular trends

Outline key design concepts to implement or consider implementing in future:

  • ALL info on mortgages should be presented on the page.
  • Create a more intuitive and logical user journey.
  • More colour variety to clarify different elements and sections.
  • Handle new and existing customers separately.
  • Improve imagery/banner - combine with information.
  • Attempt to use icons to clarify user journey and button function.
  • Important areas of the site draw attention to help users find the goals faster.
  • The user journey should be fast, easy and not confusing.
  • Make more homely + welcoming use of imagery.
  • Make more cross-platform friendly.

Design Process

At first, I started with some paper sketches and after two or so tries finally ended up with improved flow of the page. Then I recreated paper sketches as digital wireframes using axure rp7. Usually, this process is not as easy but this time I got lucky.

applied brand to my wireframes and since i did not had access to source images decided to revert back to smae styling of buttons which was on original site but separate them into two categories: new customers and current customers.

Added this completely new help section and did a couple of iterations of it. one with extra one-line description made the most sense as mortgages might get confusing.

This is something I've done as a part of my interview process for Co-operative banking.
